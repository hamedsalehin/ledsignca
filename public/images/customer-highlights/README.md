# Customer Highlights Images

This folder contains customer-style realistic showcase photos for the "Customer Highlights" section.

## Updating Highlights
To update the customer highlights photos, simply drop replacement JPG images in this folder and name them:
- `highlight_1.jpg`
- `highlight_2.jpg`
- `highlight_3.jpg`
- `highlight_4.jpg`
- `highlight_5.jpg`
- `highlight_6.jpg`
- `highlight_7.jpg`
- `highlight_8.jpg`
- `highlight_9.jpg`

Ensure they are high quality and cropped appropriately (square format works best for the grid).
